# Agency OS

Basic Next.js foundation for Agency OS.

## Stack

- Next.js App Router
- TypeScript strict mode
- Bun
- Tailwind CSS
- shadcn/ui conventions
- Supabase-ready config

## Commands

```bash
bun install
bun run dev
bun run typecheck
bun run lint
bun run build
```
